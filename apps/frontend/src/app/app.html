A clean starting point with an Angular frontend in apps/frontend and a .NET Web API in apps/backend/Pixture.Api.

Frontend

The Angular app is scaffolded with routing, SCSS, and the latest supported CLI version available during setup.

npm run start --prefix apps/frontend

Backend

The API uses ASP.NET Core controllers, OpenAPI in development, and a simple health endpoint.

dotnet run --project apps/backend/Pixture.Api
